第四节 数组

数组是一组按顺序排列的集合，集合的每个值称为元素。

1、数组的定义

2、向数组赋值

3、数组字面量方式声明

4、数组索引

与对象一样， 数组是数值的集合，不同的对象中每个数值都有一个名字，可以用名字获取对应的值，而数组的每个值都有下标值，用数组名[下标值]

5、数组长度

length属性可以设置或返回数组元素的数量，length属性的值总是比数组定义的最后一个元素的索引值大 1；

6、修改已有数组中的值

Array 可以通过索引把对应的元素修改为新的值，因此，对数组的索引进行赋值会直接修改这个数组

7、数组越界

请注意：如果通过索引赋值时，索引超过了范围，同样会引起Array大小的变化

8、数组方法indexof

数组可以通过indexof() 来搜索一个指定的元素的位置，如未找到返回-1；

9、数组concat

concat() 方法把当前的数组和另一个数组连接起来，并返回一个新的Array

10、数组push和pop

push()向数组的末尾添加若干元素，pop()则是把数组的最后一个元素删除掉；

11、数组unshift和shift

unshift向数组前面添加若干元素， shift则是把数组的第一个元素删除

12、数组slice

slice()截取数组的部分元素，然后返回一个新的Array

如果不给slice传递任何参数，他就会从头到尾截取所有元素。

13、数组sort

sort()可以对当前的Array进行排序，他会直接修改当前数组的元素位置

14、数组reverse

reverse() 把整个数组的元素给掉个个，就是反转；

15、数组join

join()方法可以把当前数组的每个元素都用指定的字符串连接起来，然后返回连接后的字符串

16、数组splice

splice()方法是修改数组的万能方法，它可以从指定的索引开始删除若干元素，然后再从该位置添加若干元素。

第一节 函数

函数是用来整合一段代码，已备后面重复使用

有时候，我们需要在不同位置使用相同的代码（重复使用）

函数的定义

小结：

  • 函数的作用：模块化，复用；
  • 函数也是一种数据类型（function）；
  • 调用函数，就是把函数体中的代码从头到尾执行一遍。

第二节 局部变量和全局变量

变量有两种作用范围：一个是在函数外，一个是在函数内

变量的作用域

变量的作用域是指哪个范围内能访问到这个变量；

在函数体内申明的变量称为局部变量，仅在函数体内可以使用；

全局变量又称为外部变量，定义在函数体外，对全局可见。

局部变量

全局变量

在函数体内部，局部变量的优先级比同名的全局变量优先级高

不var声明的变量

典型错误 不使用var申明的变量也是全局变量 （不建议这样用）

小结：

  • 能使用局部变量就不要使用全局变量
  • 【典型错误】不使用var声明的变量也是全局变量

第三节 函数传参

目的为了来扩展函数的灵活性，可以将代码中可以变化的部分提取出来，用变量代替

打招呼

一个函数可以有多个参数，调用函数时传参的个数和顺序要与函数的定义一致。

做饭

小结

参数的含义：局部变量只能在函数内部赋值，如果我想在函数外部赋值，那么可以把这变量变成参数

  • 函数定义中的参数是叫[ 形参 ] - 变量
  • 函数调用中的参数是叫[ 实参 ] - 数据
  • 函数的传参就是赋值操作
  • 函数可以有参数也可以没有
  • 如果定义了参数，在调用函数时没有传值，默认为undefined。

1、return 的含义

如果函数中没有return，则他默认返回undefined

2、retrun结束函数运行

如果在循环中使用return，不仅整个循环结束，而且 整个函数就结束了

小结：

continue 结束本次循环

break 结束整个循环

return 结束整个函数
